This a simple plugin that sends any updates from your site to your Plurk account.

== Description ==

Posts any updates to your plurk account using the Plurk API.


== Installation ==

Installation:

1. Upload wa_plurk folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Tools, Plurk Updater to set your Plurk account and other choices then that’s it.

== Frequently Asked Questions ==

= Do I need a Plurk Account =

Yes, you need an account for where it will be posted
